🔗 Reproduzir por URI

Cole aqui o URI do Spotify (música, álbum, artista ou podcast)
Como obter o URI do Spotify?
  1. Clique com o botão direito em uma música no Spotify
  2. Selecione "Compartilhar" > "Copiar URI do Spotify"
  3. Se não aparecer a opção, pressione a tecla ALT no teclado
  4. Formato: spotify:track:xxxxxxxxxxx
  5. Opcional: adicione #m:ss para definir tempo inicial
